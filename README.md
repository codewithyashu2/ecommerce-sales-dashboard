# E-commerce Sales Dashboard

End-to-end e-commerce sales analytics project — 50K+ orders analyzed 
using Python & SQL. Interactive dashboard with KPI filters, cohort 
retention analysis, and revenue trends.

---

## Tools & Technologies
- **Python** — Pandas, Matplotlib, Seaborn (data cleaning & analysis)
- **SQL** — Aggregations, Window functions, CTEs
- **HTML/CSS** — Interactive dashboard (ecommerce_dashboard.html)

---

## What This Project Does
- Analyzes 50,000+ e-commerce orders to find revenue trends
- Identifies top-performing products and categories
- Tracks customer retention using cohort analysis
- Visualizes KPIs through an interactive HTML dashboard

---

## Key Insights Found
- Peak sales occur in Q4 (Oct–Dec)
- Top 5 products contribute 38% of total revenue
- Customer drop-off is highest after first purchase
